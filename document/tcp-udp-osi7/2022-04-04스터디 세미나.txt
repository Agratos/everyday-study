리피터 : 세그먼트 간의 접속에 사용되는 것 외에 전송거리를 연장하기위해
허브 : 여러 컴퓨터를 함께 결합하여 단일 네트워크 세그먼트를 형성. 세그먼트에서 모든 컴퓨터는 서로 직접 통신 할 수 있다.
Point to Point : 네트워크에 있어 물리적으로 중개 장치를 통과하지 않고 한 지점에서 다른 지점으로 가는 채널을 말한다. 논리적으로는 두 장비간의 통신
라우터 : 컴퓨터 네트워크간에 데이터 패킷을 전송하는 네트워크 장치
L3 스위치 : 라우터를 수위치 내에 수용시켜 계층 3에서 동작시킨 장치, 빠른 동작을 하는 라우터 정도로 이해해도 무방
전이중 : 두 대의 단말기가 데이터를 송수신하기 위해 동시에 각각 독립된 회선을 사용하는 통신 방식, 고속 네트워크가 대표적

가상회선 : 데이터를 전송하기 전에 논리적 연결이 설정하는데 이를 가상회선, 정해진 시간 안이나 다량의 데이터를 연속으로 보낼때 유용, 네트워크 내의 한 도드가 다운되면 모든 가상회선을 잃게됨
데이터그램 : 패킷마다 라우터가 경로를 설정, 짧은 메시지의 일시적인 전송에 사용, 네트워크 내의 한 노드가 다운되면 다른 경로를 새로 설정
